# 🔐 Sistema de Autenticación con JWT

## 📌 Descripción
Este proyecto implementa un sistema de autenticación utilizando JSON Web Token (JWT) en Node.js. Permite validar usuarios, generar tokens y proteger endpoints mediante autorización.

---

## ⚙️ Tecnologías utilizadas
- Node.js
- Express
- jsonwebtoken
- bcryptjs
- cors
